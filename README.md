# layui-admin
Pear Admin Layui 和 laravel 后台管理

### 使用方法

env配置

```
DB_CONNECTION=mysql
DB_HOST=127.0.0.1
DB_PORT=3306
DB_DATABASE=layui-admin
DB_USERNAME=root
DB_PASSWORD=root
DB_PREFIX=xfl_

SESSION_DRIVER=redis
SESSION_LIFETIME=120
SESSION_ENCRYPT=false
SESSION_PATH=/
SESSION_DOMAIN=null

QUEUE_CONNECTION=redis

CACHE_STORE=redis
CACHE_PREFIX=cache:

REDIS_PREFIX=xfl:
REDIS_CLIENT=phpredis
REDIS_HOST=127.0.0.1
REDIS_PASSWORD=null
REDIS_PORT=6379
```
